Descargar firmware micro:bit

Personalizá el nombre Bluetooth del robot y descargá un archivo .hex listo para copiar a la micro:bit.

Usá hasta 20 caracteres ASCII. Se parchea localmente en tu navegador; no se sube ningún dato.
Listo para generar el firmware.

Cómo usarlo

  1. Escribí el nombre Bluetooth que querés ver al escanear.
  2. Descargá el archivo .hex.
  3. Conectá la micro:bit por USB y copiá el archivo descargado.
